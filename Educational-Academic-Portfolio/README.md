# CSC102 Portfolio

Educational Portfolio - Computer Science Fundamentals

## Overview

This portfolio showcases projects and exercises from CSC 102 Computer Science course. It features interactive games, string manipulation tools, and various programming exercises demonstrating fundamental computer science concepts.

## Features

- **Interactive Games**: Engaging educational games demonstrating programming concepts
- **String Manipulation**: Tools for learning string operations and algorithms
- **Visual Board Games**: Interactive board-based exercises
- **Programming Fundamentals**: Demonstrations of core CS concepts

## Educational Context

These projects were created as part of coursework for Professor Prater's CSC 102 class, demonstrating:
- Event-driven programming
- DOM manipulation
- JavaScript fundamentals
- HTML/CSS integration
- User interaction design

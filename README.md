# Nicholas M James's Portfolio

This repository contains multiple projects organized in a clean folder hierarchy.

## Repository Structure

```
.
├── index.html                  # Main entry point
├── LICENSE                     # Repository license
├── MicroLink/                  # MicroLink product
│   ├── Templates/
│   │   ├── html/              # HTML templates
│   │   ├── css/               # Stylesheets
│   │   └── js/                # JavaScript files
│   └── Assets/
│       ├── images/            # Image assets
│       ├── audio/             # Audio files
│       └── documents/         # PDF and CSV documents
└── CSC102-Portfolio/          # CSC 102 educational portfolio
    ├── Templates/
    │   ├── html/              # HTML templates
    │   ├── css/               # Stylesheets
    │   └── js/                # JavaScript files
    └── Assets/
        ├── images/            # Image assets
        └── audio/             # Audio files
```
